AI-Based Real-Time Voice Cloning & Emotion-Aware Communication System
Project Overview
This project aims to develop a real-time voice communication system integrated with:

# AI-based Voice Cloning
Speech Emotion Recognition (SER)
Live Emotion Display
Post-Conversation Emotion Analytics

The system enhances digital communication by introducing emotional awareness and personalized voice synthesis into voice chat applications.

# Features
Real-time voice chat between users
Live speech emotion detection (Happy, Sad, Angry, Neutral)
AI-based voice cloning using user voice samples
Text-to-speech in cloned voice
Post-call emotion summary and analytics dashboard
Secure and consent-based voice data handling
